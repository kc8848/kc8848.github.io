3x-ui 面板安装、配置与卸载

3x-ui 是一个支持多协议、多用户的后台管理面板，因其界面简洁和高性能而广受欢迎。本教程将带你从零开始搭建并完成安全设置。

一、 一键安装脚本

支持 Ubuntu、Debian 和 CentOS 系统。建议在全新的系统环境下运行：

# 使用官方一键脚本
bash <(curl -Ls https://githubusercontent.com)

二、 初始化设置

重要安全提示： 安装完成后，脚本会要求你设置 用户名、密码 和 端口。
1. 严禁使用默认端口 (2053)，请自定义一个 10000 以上的随机端口。
2. 确保你的防火墙（如 UFW）已放行该端口。

设置完成后，通过浏览器访问：http://你的服务器IP:自定义端口

三、 常用管理命令

在 SSH 终端输入 x-ui 即可进入管理菜单，或直接使用以下快捷命令：

x-ui # 打开管理菜单
x-ui start # 启动面板
x-ui stop # 停止面板
x-ui status # 查看运行状态

四、 SSL 证书与安全配置

  • 建议结合本站的 acme.sh 教程 申请 SSL 证书。
  • 在面板设置中开启 HTTPS，并填入证书路径。
  • 建议开启 面板根路径修改，防止被扫描工具发现。

五、 彻底卸载

如果你需要移除 3x-ui 及其所有配置，请执行：

x-ui uninstall

卸载后，建议手动清理可能残留的防火墙规则。
