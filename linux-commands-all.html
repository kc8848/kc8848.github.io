Linux 命令速查大全

未找到相关命令，请尝试其他关键词

文件与目录操作

ls

列出目录内容

ls -lah
cd

切换当前工作目录

cd /var/www
cp

复制文件或目录

cp source.txt dest.txt
rm

删除文件或目录 (慎用)

rm -rf ./temp_dir

网络与通信

ping

测试远程主机的连通性

ping google.com
netstat

显示网络连接、路由表等信息

netstat -tunlp
curl

传输数据工具，支持多种协议

curl -I https://baidu.com

系统与进程管理

top

实时显示系统中各个进程的资源占用

top
df

查看磁盘空间使用情况

df -h
chmod

修改文件或目录的访问权限

chmod 755 script.sh
