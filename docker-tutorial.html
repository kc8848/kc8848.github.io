小白篇 Docker 安装与卸载全教程

Docker 是目前全球最流行的容器引擎。它可以把你的程序及其依赖环境打包在一起，实现“一次构建，到处运行”。

一、 安装前准备

本教程适用于 Ubuntu / Debian / CentOS 等主流系统。请确保你的系统内核版本高于 3.10。

# 检查内核版本
uname -r

二、 极简一键安装（推荐）

对于小白用户，使用官方或国内镜像站提供的脚本是最简单的方案：

方案 A：使用官方脚本（海外服务器）

curl -fsSL https://docker.com | bash -s docker

方案 B：使用阿里镜像脚本（国内服务器）

curl -fsSL https://docker.com | bash -s docker --mirror Aliyun

三、 启动与验证

安装完成后，需要手动启动 Docker 服务并设置为开机自启：

# 启动 Docker
sudo systemctl start docker

# 设置开机自启
sudo systemctl enable docker

# 运行第一个容器测试
sudo docker run hello-world
💡 镜像加速提示： 国内拉取镜像较慢，建议配置阿里云或腾讯云的镜像加速器，修改 /etc/docker/daemon.json 文件即可。

四、 彻底卸载 Docker

如果你需要重新安装或不再需要 Docker，请按以下步骤彻底清理：

1. 卸载程序包

# Ubuntu / Debian
sudo apt-get remove docker-ce docker-ce-cli containerd.io

2. 删除残留文件（重要！）

注意：此操作会删除所有镜像和容器，请谨慎执行！

sudo rm -rf /var/lib/docker
sudo rm -rf /var/lib/containerd

五、 常用管理命令

  • docker ps - 查看运行中的容器
  • docker images - 查看已下载的镜像
  • docker stop 容器ID - 停止容器
