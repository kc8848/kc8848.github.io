Ubuntu/Debian 使用 UFW 配置防火墙详解

发布日期：2024年5月 • 阅读时间：5分钟

UFW (Uncomplicated Firewall) 是一个为了简化 iptables 配置而开发的防火墙工具。它是 Ubuntu 默认的防火墙配置工具，非常适合新手保护服务器安全。

1 检查 UFW 状态

在开始之前，先检查防火墙是否已经开启：

sudo ufw status verbose

如果输出 Status: inactive，说明防火墙目前处于关闭状态。

2 设置默认策略

通常最安全的做法是：禁止所有入站规则，允许所有出站规则。

sudo ufw default deny incoming
sudo ufw default allow outgoing
⚠️ 极其重要： 在启用防火墙之前，请务必先允许 SSH 连接，否则你会被锁在服务器外面！

3 允许常用端口

根据你的需求开启对应的服务端口：

# 允许 SSH (默认22端口)
sudo ufw allow ssh

# 允许 HTTP (80端口) 和 HTTPS (443端口)
sudo ufw allow http
sudo ufw allow https

# 允许特定范围的端口 (如 3000 到 3100)
sudo ufw allow 3000:3100/tcp

4 启用防火墙

确认 SSH 已允许后，正式开启防火墙：

sudo ufw enable

系统会提示此操作可能会中断 SSH 连接，输入 y 继续即可。

5 其他常用命令

如果你需要删除某条规则或关闭防火墙：

# 删除规则 (例如删除允许 HTTP 的规则)
sudo ufw delete allow http

# 禁用防火墙
sudo ufw disable

# 重置所有规则
sudo ufw reset
