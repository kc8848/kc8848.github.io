使用 Bench.sh 一键测试服务器性能

什么是 Bench.sh？

由秋水逸冰（Teddysun）开发的经典脚本。它可以一键查看服务器的系统信息、测试全球主要节点的下载速度以及硬盘的 I/O 读写速度。是买到新 VPS 后的“必跑”项目。

一、快速执行脚本

你不需要下载任何文件，直接在终端复制并粘贴以下命令即可：

# 使用 wget 执行
wget -qO- bench.sh | bash

# 或者使用 curl 执行
curl -Lso- bench.sh | bash

二、测试结果解读

脚本运行结束后，你会看到三部分主要数据：

区块 说明
System Info 包含 CPU 型号、核心数、主频、内存、SWAP、运行时间及内核版本。
I/O Speed 硬盘读写速度。三次测试的平均值越高，代表你的硬盘性能越好（SSD通常在200MB/s以上）。
Node Name 包含全球各地的下载速度测试，关注 "Download Speed" 列。

三、进阶参数

如果你只想查看特定信息，可以使用参数：

# 仅显示系统信息和 I/O 测试
wget -qO- bench.sh | bash -s info

💡 小贴士

1. 建议在服务器负载较低时运行，以获得最准确的 I/O 和速度数据。
2. 如果你的服务器在海外，国内节点的下载速度受线路（如 CN2 GIA）影响较大。

四、常见问题

问：为什么提示命令未找到？
答：请先安装 wget 或 curl。例如在 Ubuntu 上执行 sudo apt install wget -y。
