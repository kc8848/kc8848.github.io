s-ui 面板小白全教程

s-ui 是一款极其轻量、简洁的交互式管理面板。相比于功能繁杂的 3x-ui，s-ui 更加专注于核心功能的稳定性，非常适合配置较低的 VPS 服务器。

注意： 安装前请确保你的系统防火墙已开放 54321 端口（这是 s-ui 的默认访问端口）。

一、 一键安装脚本

在你的 SSH 终端粘贴以下命令：

# 使用官方安装脚本
bash <(curl -Ls https://githubusercontent.com)

二、 登录与初始化

默认信息：

1. 浏览器访问：http://服务器IP:54321

2. 初始用户名：admin

3. 初始密码：admin

强烈建议： 登录后立即在面板设置中修改默认用户名、密码以及访问端口，以防被暴力破解。

三、 常用快捷命令

在 SSH 中输入 s-ui 即可调出功能菜单。常用子命令如下：

s-ui start # 启动面板
s-ui stop # 停止面板
s-ui restart # 重启面板
s-ui status # 查看运行状态

四、 面板设置建议

  • 公钥与私钥： 建议配合 SSL 证书开启 HTTPS 访问。
  • 端口放行： 如果配置了节点，记得在 UFW 防火墙中打开对应的节点端口。
  • 版本更新： 在 s-ui 菜单中选择更新功能，保持内核处于最新稳定版。

五、 彻底删除

如果你想卸载 s-ui，只需在 SSH 中执行：

s-ui uninstall

该命令会自动停止面板服务并清理相关系统文件。
